SD女仆–系统清理工具5.0.5 APK + MOD（已修补）Android

 2020-12-23 855 words No tag

没有人是完美的，Android也不是。 您已删除的应用，有时会留下数据。 系统会不断创建您真正不需要的日志，崩溃报告和调试文件。 SD卡正在收集您无法识别的文件和目录。 让我们不要继续…

为什么不请女仆把它清理干净呢？ SD女仆将帮助您使设备保持整洁，整洁:-)！

试试看，您将找不到更彻底的清洁工具！

选择一个选项卡，按开始，然后单击单个项目删除或按清除所有按钮。就这么简单。

有多种可用功能：

  • 资源管理器是成熟的文件管理器，可使用它来浏览Android文件。
  • 如果知道要查找的文件，则可以使用“搜索器”。
  • CorpseFinder在您的设备中搜索孤立的项目，并将其与已安装的应用程序列表进行比较。
  • AppControl可让您冻结，重置和删除应用程序（甚至系统应用程序）。
  • SystemCleaner会扫描您的设备并过滤包含不必要文件的目录。您甚至可以创建自己的过滤器！
  • 您可以优化和缩小肿的数据库，以加快访问速度和释放空间。
  • 查看最大的文件，找出占用所有空间的文件。
  • 查找最近X分钟内已修改的文件。
  • 和更多…

可以通过购买和安装解锁器（SD Maid Pro）来解锁一些其他功能以及舒适性和便利性功能。

SD女仆一直在非常仔细地进行开发和设计。 由于有大量的Android设备，SD Maid可能无法在您的设备上完美运行，在这种情况下，如果您有机会与我联系以进行修复，我将不胜感激。 如果您有任何疑问或疑虑，请给我发送邮件（support@darken.eu）。我通常会很快做出回应，并会很乐意为您提供帮助。
